# TCP/IP Protocol Suite

## Overview
TCP/IP is the foundational protocol stack of modern networking. It defines how data is packaged, addressed, transmitted, routed, and received.

## Key Protocols
- **TCP (Transmission Control Protocol)**  
  Reliable, connection‑oriented communication. Ensures packets arrive in order.

- **UDP (User Datagram Protocol)**  
  Fast, connectionless communication. No guarantee of delivery.

- **IP (Internet Protocol)**  
  Handles addressing and routing of packets across networks.

- **ICMP (Internet Control Message Protocol)**  
  Used for diagnostics (e.g., ping, traceroute).

## Common Use Cases
- TCP → web browsing, email, file transfer  
- UDP → streaming, gaming, VoIP  
- ICMP → troubleshooting  
- IP → routing everything

## Useful Commands
- `ping 8.8.8.8`
- `tracert google.com`
- `netstat -an`
- `ipconfig` / `ifconfig`
